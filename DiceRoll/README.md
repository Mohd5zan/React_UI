# DiceRoll

DiceRoll is a simple React-based UI application that simulates rolling a dice.  
Each roll generates a random number between 1 and 6 and updates the UI accordingly.

## Tech Stack

- **React** – Component-based UI development  
- **JavaScript (ES6+)** – Application logic and state handling  
- **HTML5** – Markup structure  
- **CSS3** – Styling and layout  
- **npm** – Dependency management  

## Project Setup

```bash
git clone https://github.com/Mohd5zan/React_UI.git
cd React_UI/DiceRoll
npm install
npm start
